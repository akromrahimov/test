Анализ банковских депозитов

Стек: SQL • Microsoft Excel • Power BI

О проекте:
 В рамках проекта проведён анализ данных клиентов банка с целью оценки эффективности привлечения клиентов к оформлению депозитов.
Анализ выполнен с использованием SQL, Microsoft Excel и Power BI. На основе данных были рассчитаны ключевые показатели эффективности (KPI), проведён сравнительный анализ различных сегментов клиентов и разработан интерактивный дашборд для визуализации результатов.

Технологии:

SQL — анализ и агрегация данных;
Microsoft Excel — подготовка данных;
Power BI — визуализация и создание интерактивного дашборда.

Основные задачи:
* Рассчитать общее количество клиентов и оформленных депозитов.
* Определить общий коэффициент конверсии.
* Проанализировать конверсию по профессиям клиентов.
* Сравнить эффективность различных каналов связи.
* Исследовать изменение конверсии по месяцам.
* Построить интерактивный дашборд для анализа результатов.
* При выполнении анализа использовались агрегатные функции (COUNT, SUM, ROUND), CASE WHEN, GROUP BY, общие табличные выражения (CTE) и оконные функции (RANK()).

Ключевые результаты:
* Общая конверсия составила 11%.
* Канал связи Cellular показал более высокую эффективность по сравнению с телефонными звонками.
* Наибольшая конверсия наблюдается среди клиентов категории Student.
* Анализ показал различия в эффективности привлечения клиентов в зависимости от месяца проведения кампании.

Дашборд
Дашборд отображает основные KPI и позволяет анализировать результаты с помощью интерактивных фильтров. Представлены показатели общей конверсии, эффективности каналов связи, распределения клиентов по профессиям и динамики конверсии по месяцам.
